## 🚀 Quick Start

### 1. Install
```bash
pip install cbridge-agent
```

### 2. Initialize
```bash
cbridge init
```

### 3. Add Watch Directory
```bash
cbridge watch add ~/Documents/MyProjects
```

### 4. Search
```bash
cbridge search "your query"
```
