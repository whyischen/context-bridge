# ContextBridge Release Note

## v0.1.19 - Initial Release

### 🚀 New Features

**Core Capabilities**
- Multi-format document parsing (PDF, Word, Excel, PPTX, Markdown) with configurable strategies
- Embedded ChromaDB vector database with automatic indexing
- Intelligent search algorithm with adaptive ranking and relevance optimization
- Real-time file monitoring and auto-sync using Watchdog
- 100% local-first architecture with privacy protection

**Document Processing**
- Dual PDF parsing strategies: lightweight MarkItDown (default) and high-precision Docling (optional)
- Composite parser system supporting multiple document formats
- GTE-Small-Zh ONNX INT8 quantized embedding model for efficient vector search
